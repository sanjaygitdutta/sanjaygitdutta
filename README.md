## About

<img align="right" alt="Night coding animation" src="https://raw.githubusercontent.com/AVS1508/AVS1508/master/assets/Night-Coding.gif" width="280" />

I'm an **AI Systems Engineer** who crossed over from **3+ years of professional derivatives trading** — bringing rare production instincts to intelligent systems.

I build **cloud-native, end-to-end AI pipelines**: multimodal inference, LLM reasoning, distributed backends, and serverless deployments on **AWS + GCP**. Wired for high-stakes, real-time decision environments.

**My operating system**
- **Find the right use case** — where ROI is real, not vibes
- **Design the system** — data, evals, cost, latency, safety
- **Ship to production** — monitoring, governance, iteration
- **Drive adoption** — workflow fit, enablement, feedback loops

**Currently**
- BS–MS in **AI & Cybersecurity** @ **IIT Patna** (2025–2030)
- Building on **AWS** (Lambda, API Gateway, CDK) + **Google Cloud Run**, exploring **RAG + Agents** and **MCP**

## Featured Projects

### 🚨 SankatMitra — Cloud-Native Emergency Route Optimization
> `AWS Lambda` `RNN/LSTM` `FastAPI` `Docker` `AWS CDK`  
> 🏆 Shortlisted — AI for BHARAT Hackathon

Fully serverless emergency corridor engine — ingesting real-time GPS data, predicting route times with RNN/LSTM, and generating dynamic corridors at low latency. Hardened with GPS spoofing detection, JWT-secured APIs, multilingual GenAI public alerts, and full IaC provisioning via AWS CDK.

---

### 🍳 Fridge Chef AI — Real-Time Conversational Kitchen Assistant
> `Gemini 2.0 Flash Live API` `FastAPI` `Google Cloud Run` `Firestore` `WebRTC`  
> Gemini Live Agent Challenge Hackathon

Live AI kitchen agent over WebSockets — fully conversational and interruptible, with zero-latency bidirectional PCM audio. Gemini Vision for zero-shot fridge scanning → auto digital pantry; Gemini 1.5 Flash for structured JSON recipe generation. Full stack deployed via IaC on Cloud Run.

---

### 📈 Fantastic-FinanceAI — Distributed Multimodal Financial Intelligence Engine
> `FastAPI` `Celery` `Redis` `PostgreSQL` `Docker`

Production-grade distributed backend fusing video, audio, text, and chart signals for financial anomaly detection — orchestrated asynchronously via Celery + Redis. Modular microservices behind REST inference APIs; full multi-service stack containerized with Docker Compose and end-to-end test coverage.

---

### 👗 FashionAI — Multimodal LLM Recommendation Engine
> `Python` `FastAPI` `Streamlit` `Gemini Vision`  
> Gemini 3 Hackathon

Fused Gemini Vision image understanding with LLM contextual reasoning for real-time outfit recommendations — from image upload → prompt orchestration → live Streamlit UI.
